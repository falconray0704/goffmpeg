## Pre-requirments
Linux:
- autoconf
- autoconf-archive
- automake
- pkg-config
- yasm

Windows:
Unsupported yet

## Installation

	git clone https://github.com/zhangpeihao/goffmpeg.git goffmpeg
	cd goffmpeg
	./build.sh
